# FUTURE_CS_02
This project involves analysing phishing emails to identify common attack patterns, classify risk levels, and understand how social engineering techniques are used to trick users. The goal is to improve awareness and demonstrate basic cybersecurity analysis skills
Tools used:
Email samples (provided in assignment / GitHub repositories)
Web browser (for safe inspection of URLs and domains)
Microsoft Word / Google Docs (for report writing)
GitHub (for version control and submission)
Analysis Approach:
1) Phishing email samples were collected
2) The sender email address and domain were checked to identify spoofed or suspicious identities
3) The email body was reviewed for:
Urgency or fear-based language
Requests for sensitive information (passwords, OTPs, banking details)
Generic greetings and poor personalization
All URLs were examined to detect:
Mismatched or fake domains
Each email was classified as:
Phishing
Findings were documented in a structured report including:
Indicators of phishing
Risk level
Explanation of attack method
Prevention recommendations
